# USA Family Sample

Patel household is a two-income family ledger with a mortgage and childcare costs.

## What This Shows

- Two payroll streams with retirement contributions
- Mortgage split between interest expense and principal reduction
- Household spending for groceries, utilities, daycare, and 529 savings

## Commands

```bash
./bin/cpa-check main.beancount
./bin/cpa-fava main.beancount 5000
```

This sample is for illustration only.
