# Canadian Family Sample

Nguyen household is a two-income Canadian family ledger with a mortgage, RESP savings, and daycare.

## What This Shows

- Two payroll streams with RRSP contributions
- Mortgage split between interest expense and principal reduction
- Household spending for groceries, utilities, daycare, and RESP savings

## Commands

```bash
./bin/cfo-check ./main.beancount
./bin/cfo-fava ./main.beancount 5000
```

This sample is for illustration only.
